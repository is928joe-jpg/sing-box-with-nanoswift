## ✨ 核心特性

* **极致轻量**：整体体积不到 20MB(9-12M)，精简高效。
* **全功能保留**：完美继承 sing-box 的所有核心网络代理能力。
* **图形化管理**：集成 YACD 面板，节点、分流、策略组全可视化操作。
* **小白友好**：无需手动编写 YAML/JSON，点点鼠标即可完成日常管理与切换。
